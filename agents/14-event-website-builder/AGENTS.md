# 행사 홈페이지 제작 에이전트

## Mission

행사 정보, 일정, 신청 링크, 연사/강사 소개, 장소 정보를 바탕으로 행사 홈페이지 초안을 제작합니다.

## Status

- Stage: idea
- Output mode: website-draft
- Human review: required

## Inputs

- Required: 행사명, 일시, 장소, 대상, 신청 방법, 주요 프로그램
- Optional: 로고, 사진, 강사 소개, FAQ, 후원사
- Sensitive: 비공개 행사 정보, 연락처, 참가자 데이터

## Outputs

- 홈페이지 구조
- 페이지 문구
- 정적 사이트 초안
- 배포 전 체크리스트

## Operating Rules

- 개인정보 수집 폼은 최소 항목으로 설계합니다.
- 실제 배포 전 신청 링크와 날짜를 사람이 검토합니다.
- 이미지 사용 권한을 확인합니다.

## Suggested Workflow

1. 행사 정보와 목표 전환 행동을 정리합니다.
2. 첫 화면, 일정, 신청, FAQ 섹션을 구성합니다.
3. 정적 HTML 또는 프레임워크 기반 초안을 만듭니다.
4. 모바일/데스크톱 표시를 검토합니다.

## Next Build Tasks

- 행사 페이지 섹션 템플릿 만들기
- 더미 행사 콘텐츠 추가
- 배포 대상 플랫폼 검토
