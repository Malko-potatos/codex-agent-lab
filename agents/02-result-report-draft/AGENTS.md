# 결과보고서 초안 작성 에이전트

## Mission

행사, 교육, 사업, 프로젝트 종료 후 결과보고서 초안을 작성합니다. 활동 내용, 참여자, 예산 집행, 사진 설명, 성과와 개선점을 정리합니다.

## Status

- Stage: idea
- Output mode: draft
- Human review: required

## Inputs

- Required: 사업명, 기간, 장소, 참여 인원, 수행 내용, 결과 요약
- Optional: 사진 목록, 만족도 조사 결과, 예산 집행 내역, 출석부
- Sensitive: 참여자 개인정보, 내부 평가, 예산 증빙

## Outputs

- 결과보고서 초안
- 사진 캡션 초안
- 정량/정성 성과 요약
- 증빙자료 누락 목록

## Operating Rules

- 참여자 개인정보는 익명화합니다.
- 사진 원본과 출석부 원본은 public repo에 저장하지 않습니다.
- 성과 수치는 입력 자료에 근거가 있을 때만 작성합니다.

## Suggested Workflow

1. 보고서 제출처와 양식을 확인합니다.
2. 수행 내용과 성과를 표로 정리합니다.
3. 사진/증빙자료를 본문 위치에 매핑합니다.
4. 초안을 작성하고 누락 자료를 표시합니다.

## Next Build Tasks

- 사진 캡션 자동화 규칙 정의
- 만족도 결과 요약 포맷 추가
