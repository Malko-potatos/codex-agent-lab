# Security and Privacy Policy

이 저장소는 public repository로 공개될 수 있으므로 민감정보를 저장하지 않는 것을 기본 정책으로 합니다.

## 저장 금지

- 직원 급여명세서, 주민등록번호, 계좌번호, 연락처
- 고객사 계약서, 견적서 원본, 세금계산서 원본
- 제안요청서, 과업지시서, 기존 제안서 원문
- 네이버웍스 export, 첨부파일, 토큰, 세션
- 네이버카페 로그인 쿠키, 크롤링 원문, 회원 정보
- 기자 연락처 DB, 이메일 발송 목록
- API key, OAuth secret, service account key

## 허용

- 익명화된 샘플 데이터
- 사람이 직접 만든 더미 문서
- 공개 가능한 템플릿
- 실행되지 않는 설계 메모
- dry-run 전용 예제 코드

## 외부 작업 승인 기준

아래 작업은 사람이 명시 승인해야 합니다.

- 이메일, 메신저, 네이버웍스 메시지 발송
- 보도자료 배포
- 블로그, 인스타그램, 홈페이지 게시
- 로그인 세션을 사용하는 크롤링
- 외부 스토리지에 파일 업로드
- 실제 계약/견적/급여/세금 문서 생성 및 발송

## 설계 원칙

- 기본값은 읽기 전용입니다.
- 기본 출력은 초안입니다.
- 외부 전송은 dry-run으로 먼저 구현합니다.
- 모든 자동화에는 중단 지점과 검토 지점을 둡니다.
- public repo에는 운영 데이터 대신 schema, mock, template만 둡니다.
